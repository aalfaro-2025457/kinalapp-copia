# kinalapp
Aplicacion web de gestion de compra y venta de productos
Utilizando una Api REST

## Tecnologias Utilizadas
****

* **Java 21**
* **SpringBoot 4.0.2**
* **SpringBoot Web**
* **MySQL** (Sistema Gestor de Base de Datos)
* **Maven** (Gestor de Dependencias)

## Requisitos Previos
****
* Jdk 17 o superior
* Maven Instalado
* Instancia Activa de MySQL
